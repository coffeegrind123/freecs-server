FreeCS

Open-source Counter-Strike 1.5 reimplementation — powered by FTEQW.

  • Complete CS 1.5 gameplay: bomb defusal, hostage rescue, assassination, and more.
  • Runs on the FTEQW engine with patched dpmaster server browser support.
  • Includes CS 1.5 maps, models, sounds, and Half-Life base game assets.
  • Built-in bot support with waypoints for 55+ maps.
  • Reports to ms.cs16.net master server — find servers in the browser.
